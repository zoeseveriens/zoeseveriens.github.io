Lowlands

Een responsive redesign van de Lowlands website voor het vak front-end development.

Jaar: 2019

Vak: Front-end Development

Categorie: Front-end Development, Webdesign

Opdracht

Voor het vak front-end development was het de bedoeling om een gekozen website na te maken en eventueel te redesignen. Ik had gekozen voor de Lowlands website, omdat deze nogal druk en chaotisch is en ik had hier toevallig al een redesign voor gemaakt bij een ander vak. Door middel van flexbox en media queries heb ik de website responsive kunnen maken zodat de website op verschillende devices goed bruikbaar is. De meeste aandacht is besteed aan HTML en CSS maar er is wel een klein beetje Javascript toegepast wat dit project uitdagender maakte.
